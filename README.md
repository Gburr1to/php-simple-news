# PHP Simple News 📰

## A simple portal for publishing and reading news, built with PHP.
### Key Features

    - Read News

    - Registration & Login

    - Publishing articles: title, abstract, and content via a simple form.

    - Security: The system automatically logs out users after 30 minutes of inactivity.

## How to set up (Installation)

    - Database: Create a new database in MySQL (phpMyAdmin), import new table from news.sql. 

    - Database Connection: In header.php, check $conn. Ensure the newly created database, username, and password are correct.

    - Path Configuration: Open the root.php file and change the path to match your local setup (e.g., /php-simple-news/).
